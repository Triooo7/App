//Java 1.8 //

Running the Application


1.Run the Database.sql file to setup the database.
2.Few employee details are already mentioned in the employee table. You can also enter any details you want.
3.Enter your own username and password of mysql database in config.yml.
4.Run backend_main.jar to start the server using this command in commandprompt:
java -jar backend_main.jar server config.yml

Now you can view the application through localhost:8080.
